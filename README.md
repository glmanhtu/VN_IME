Gõ tiếng Việt thông minh trên Sublime Text

Cài đặt locally:
```
ln -s ~/repos/fingers-sublime ~/Library/Application\ Support/Sublime\ Text/Packages/fingers-sublime
```
Nhấn phím `esc` để bật tắt chế độ gõ tiếng Việt.
